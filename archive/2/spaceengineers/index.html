ENPaulNet Space Engineers

Space Engineers is an open world construction and simulation game where you can build everything from a bicycle to an interplanetary warship. You can learn more about it on the Official Website, the Official Wiki, or the Player-Made Wikia.

The Space Engineers Server

The ENPaulNet runs a Space Engineers Dedicated Server:



spaceengineers.apps.enpaul.net

The ENPaulNet Server is a creative mode planets server that requires DX11 clientside. Server updates are pushed a day or two after the weekly game update.

For connection instructions see the Security section.

Information on the mods available on the server is in the Mods section.

Steam Server Mods

Connections and Security

We want the ENPaulNet Space Engineers Server to remain a fun enviroment for all players



For security and to protect against griefers the server uses an allow-by-exception security policy; without being added to the whitelisted Steam Group your connection will be refused. To request server access please request to be added to the group.
Information on the mods available on the server is in the Mods section.

There is currently no support for unauthenticated users on the ENPaulNet Space Engineers Server
