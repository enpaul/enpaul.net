The ENPaul Network (ENPN) provides a number of services for entertainment, car parts, data management, and security. While the full list of services and applications hosted on the ENPN is constantly in flux, the services included here are persistent and should be considered reliable.

Please report any service outages, interrupts, or problems by emailing the administrator.

Public Links

  Virtual Private Networking


A Virtual Private Network (VPN) allows for secure access to an internal network. Most ENPN Services require a connection through the VPN.

The ENPN VPN system uses the open-source VPN server software from the Softether Project. The secure L2TP/IPSec tunnel uses Active Directory authentication and AES128 encryption for security.

  • More Information

  Active Directory


Microsoft Active Directory Domain Services (ADDS) is the backbone of all ENPN authentication, including RADIUS, Kerberos, and LDAP.

The ENPN Domain makes use of ADDS distributed domain authentication and Group Policy system management. This allows for automation and remote management of large networks by a small number of administrators.

Connecting to the ENPN Domain requires VPN access

  Network Storage


Centralized network storage ensures that data is accessible and redundant within the network infrastructure. This protects against data loss, promots collaboration, and allows for easy organization.

The primary ENPN datastore is a RAID 5 array of Western Digital Red drives, while the secondary datastore uses twin Western Digital Green drives in a mirror configuration.

The ENPN Storage Drive uses the open-source Nextcloud software project to provide a publicly accessible web interface for accessing data resources.

  Plex Media Server


The Plex Media Server project allows administrators to make their personal media collections available to friends and family. The ENPN Plex Media Server is available both at the link below and directly through Plex.TV.

Plex Media Server does not use Active Directory Authentication
