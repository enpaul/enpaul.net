The ENPN Virtual Private Networking service uses an L2TP/IPSec tunnel secured with AES128/SHA256 for intranet connectivity. This protocol is natively supported by most devices and operating systemes, but may be blocked by some firewalls.

NOTE: Computers linked to the ENPN Omni Domain are automatically provisioned with the VPN profile and do not need to be configred.


Choose your client platform below:

Connecting from Windows



NOTE: For security reasons, remote connectivity from Windows clients running Windows Vista, Windows XP, or older is not supported


Adding a connection through the Windows 10 Settings App
  1. From the Start Menu, open the Settings app
  2. In the main Settings app, select Network & Internet, then from the right hand menu select VPN
  3. Select Add a VPN connection and complete the required fields with the following information:
    • Set "VPN Provider" to Windows (built-in)
    • Set "Connection name" to ENPN VPN or similar
    • Set "Server name or address" to vpn.enpaul.net
    • Set "VPN type" to L2TP/IPSec with pre-shared key
    • Set the "Pre-shared key" field equal to the VPN key provided to you. If you have not been provided with the VPN key please email info@enpaul.net
    • Leave "Type of sign-in info" set to User name and password
    • Use the "Username" and "Password" fields to enter your ENPN Omni Domain login credentials.
    • Opitonally, select "Remember my sign-in info"
  4. Click Save and Connect
Adding a connection through the Control Panel (Windows 8.1, 8, and 7)
  1. From the Start Menu either click or search for Control Panel
  2. Click Network and Internet, then click Network and Sharing Center
  3. Click Set up a new connection or network, select Connect to a workplace, then click Next
  4. Select Use my internet connection (VPN)
  5. Set the following parameters:
    • Set "Internet Address" to vpn.enpaul.net
    • Set "Destination name" to ENPN VPN or similar
    • Uncheck "Use a smart card"
    • Check "Remember my credentials"
    • Uncheck "Allow other people to use this connection"
  6. Click Create
  7. From the Network and Sharing Center select the newly created connection and click Connect
  8. When prompted, enter your ENPN Omni Domain credentials

Connecting from Android



NOTE: These instructions assume Andorid version 5.0 or newer. Android version 4.1 or newer is required.


  1. Open the Settings app from either the app drawer or the pulldown menu
  2. In the Wireless & Networks section select More
  3. Select VPN
  4. In the upper right corner, select the Plus () sign
  5. In the Edit VPN profile dialog set the following parameters:
    • Set "Name" to ENPN VPN or similar
    • Select L2TP/IPSec PSK for "Type"
    • Set "Server address" to vpn.enpaul.net
    • Set "IPSec pre-shared key" to the VPN key provided to you. If you have not been provided with the VPN key please email info@enpaul.net
  6. Select Save
  7. Select the newly created VPN connection from the list of VPN connections to connect
  8. When prompted enter your ENPN Omni Domain credentials
  9. Optionally, select Save account information
  10. Select Connection
